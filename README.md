# CODE DE BOTIGA
------------------

Aquest codi té la funció de calcular preus segons els tipus de categories de la botiga, finalment aplicant un descompte, donant el preu total que segons l'edat se'ls pot aplicar un val de descompte.


1. VARIABLES AMB CONSTANTS (Evitar Magic Numbers)

Tenim només començar quatre variables inicials encarregades d'indicar diferents valors.

Cal destacar que les variables i noms són els encarregats de poder indicar correctament que significa cada valor.


2. FUNCIÓ AJUST SEGONS CATEGORIA

El preu l'hem de calcular segons la categoria que el client hagi escollit, llavors amb aquesta funció, no només agilitzem el procés, sinó que també deixem el codi principal més net i entenedor.


3. FUNCIÓ PRINCIPAL

L'encarregada de donar-nos el preu final. Si ens fixem dintre del codi veiem que tenim una condició que diu que, segons l'edat podem obtenir un descompte. Al final ens imprimeix el preu final de tot.


4. EXTRA

Al final del codi es pot provar el seu funcionament.
